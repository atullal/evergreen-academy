Global Accessibility & Translation Guide

Evergreen Digital Academy is built to be accessible to older adults in every corner of the world. Because many older readers are new to browsing, we rely on automatic browser translation tools rather than complex selectors that can fail or confuse old devices.

How to translate these lessons in your language

This website is built with clean, semantic web code. This makes it instantly compatible with the automatic translation features inside modern web browsers like Google Chrome, Safari, and Microsoft Edge.

  1. In Google Chrome (Phone or Computer): Look at the top right of your address bar for the translation symbol (it looks like a small G or folded paper), or tap the menu (three dots) and choose "Translate...".
  2. In Safari (iPhone or iPad): Tap the "aA" button on the left side of your address bar, then tap "Translate to [Your Language]".
  3. In Microsoft Edge: The browser will automatically ask to translate the page if your device is set to another language. If not, tap the translate icon in the address bar.

Accessibility Standards

  • Typography: Large, clean font-face sizing options that can scale up dynamically using the "A / A / A" controls.
  • Contrast: High-contrast color values that pass strict WCAG AA standards for low vision.
  • RTL Support: Full Right-to-Left formatting automatic scaling for languages like Arabic, Hebrew, and Farsi.
  • Zero-Bloat: No heavy web libraries, meaning pages load instantly even on weak 3G networks and 10-year-old hardware.
